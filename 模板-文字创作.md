# 小说创作规范（通用模板）

> 说明：本规范用于明确小说创作的核心要求，供AI生成内容时遵循。你可根据实际创作需求（如题材、受众、篇幅等）调整、增补或删减条款，确保AI输出内容符合你的预期。

## 一、核心基础信息

1. **题材与子类型**：明确核心题材（如科幻、悬疑、言情、武侠、玄幻、现实、历史等），若有子类型需进一步细化（例：科幻-硬科幻-太空探索；言情-都市甜宠-职场恋爱；悬疑-本格推理-校园凶案）。
2. **目标受众**：明确读者群体（如青少年、年轻女性、悬疑爱好者、科幻迷等），需适配受众的阅读习惯、情感需求及认知水平（例：面向青少年的内容避免过于血腥暴力，面向职场人的内容可融入职场痛点）。
3. **篇幅与结构**：确定小说篇幅（短篇：1-3万字；中篇：3-10万字；长篇：10万字以上）；明确结构形式（如线性叙事、多线并行、倒叙/插叙结合、单元剧形式等）；若为长篇，可初步规划卷/章划分逻辑（例：按情节阶段分卷，每卷聚焦一个核心冲突）。
4. **核心主题**：明确小说想要传递的核心思想或情感（如探讨人性善恶、歌颂爱情坚守、反思科技伦理、展现时代变迁等），确保所有情节、人物行为均围绕核心主题展开。

## 二、世界观设定（若为虚构题材，需详细说明；现实题材可简化）

1. **背景环境**：
        时间维度：明确故事发生的时间（如古代-唐朝贞观年间、现代-2025年、未来-3000年星际时代、架空时间线-某王朝统治时期）。
2. 空间维度：描述核心场景（如地球-某座现代都市、异世界-魔法大陆、太空-星际联盟空间站、乡村-某南方古镇），说明场景的关键特征（如都市的繁华与冷漠、魔法大陆的种族分布、空间站的运行规则）。
3. **核心规则**：
        虚构体系规则（若有）：详细说明世界观内的特殊体系（如魔法、异能、修仙、机甲、超能力等）的设定逻辑——来源（天生拥有、后天修炼、科技赋予等）、等级划分（例：修仙分炼气、筑基、金丹等境界）、使用限制（如消耗能量、有冷却时间、存在副作用）、核心原理（如魔法依托元素之力、异能与基因变异相关）。
4. 社会/制度规则：描述世界观内的社会结构（如封建帝制、联邦共和、种族联盟、帮派割据等）、核心制度（如法律条文、等级制度、资源分配规则、婚姻制度等）、人际关系准则（如尊卑秩序、族群矛盾、合作模式等）。
5. 自然/物理规则（若有修改）：若世界观违背现实物理/自然规则，需明确说明（如重力减半、存在魔法元素、生物可永生等）；若遵循现实规则，可标注“符合现实世界物理/自然规律”。
6. **关键元素**：列出世界观内的核心元素（如特殊物品、神秘组织、核心资源、禁忌事物等），说明其功能、意义及相关背景（例：核心物品“时光沙漏”可回溯30分钟，是反派争夺的关键；神秘组织“守夜人”负责守护世界免受超自然力量侵害）。

## 三、人物设定

1. **核心人物（主角/女主角/男主角）**：
        基础信息：姓名、性别、年龄、外貌特征（需突出辨识度，例：“男主留着寸头，左眉有一道浅疤，常年穿黑色工装”）、身份背景（如普通职员、皇室后裔、修仙弟子、侦探等）。
2. 性格特点：明确核心性格（如乐观坚韧、隐忍腹黑、温柔善良、桀骜不驯），需包含性格反差或成长空间（例：表面冷漠的女主内心渴望温暖，前期懦弱的男主后期变得勇敢），避免人物过于扁平化。
3. 核心动机：驱动人物行动的核心需求（如复仇、寻找真相、守护亲人、实现理想、追求自由等），动机需合理且贯穿全文。
4. 能力/弱点：列出人物的核心能力（如擅长推理、拥有火系魔法、精通格斗、具备商业头脑）、致命弱点（如恐高、重视的人易被要挟、能力有致命副作用、性格冲动）。
5. 成长弧线：明确人物的成长变化（例：从自私自利到懂得牺牲，从迷茫无助到目标坚定，从仇恨缠身到学会释怀），说明推动其成长的关键事件（如经历重大挫折、遇到重要的人、发现真相）。
6. **重要配角（男二/女二/关键盟友/导师等）**：
        基础信息：姓名、性别、年龄、身份、与主角的关系（如挚友、对手、导师、家人）。
7. 核心功能：明确配角在故事中的作用（如辅助主角成长、推动情节发展、制造冲突、提供关键信息、作为主角的情感寄托等）。
8. 性格与特点：具备鲜明的性格特征（避免工具人属性），可搭配1-2个关键特点（如话痨但可靠、高冷却心软、老谋深算却心怀正义）。
9. **反派角色**：
        基础信息：姓名、性别、年龄、身份、表面形象（如伪善的慈善家、冷酷的反派首领、隐藏在主角身边的敌人）。
10. 反派动机：明确反派作恶的原因（如复仇、权力欲、扭曲的理想、被迫无奈、对世界的怨恨等），避免纯粹的“恶”，可赋予合理的背景故事（如童年创伤、被背叛经历）增强角色立体感。
11. 能力与手段：列出反派的核心能力（如智商极高、拥有强大的力量、掌控庞大的资源、擅长阴谋诡计）、常用手段（如挑拨离间、制造危机、暴力压迫、利用他人）。
12. 与主角的关系：明确反派与主角的冲突核心（如理念对立、利益争夺、个人恩怨），说明两者之间的博弈逻辑。
13. **人物关系网**：梳理核心人物与其他角色的关系（如朋友、敌人、恋人、亲属、上下级），明确关键关系的张力（如相爱相杀、互相利用、生死与共），确保人物互动符合逻辑。

## 四、情节要求

1. **核心冲突**：
        主线冲突：明确故事的核心矛盾（如主角与反派的对抗、主角追求目标过程中的阻碍、不同理念之间的碰撞、个人与时代的冲突等）。
2. 支线冲突：规划关键支线的冲突（如主角与配角的矛盾、配角之间的利益争夺、小范围的危机事件等），支线冲突需服务于主线冲突或人物成长。
3. **情节节奏**：
        整体节奏：明确节奏风格（如快节奏、慢节奏、张弛有度），例：“悬疑小说需保持快节奏，每章结尾留悬念；言情小说可适当放缓节奏，侧重情感细节描写”。
4. 关键节点：规划故事的核心情节节点（按开端-发展-高潮-结局梳理）：
           开端：如何引入主角、世界观及核心矛盾（例：“主角在一场意外中发现自己拥有异能，同时被神秘组织追杀，引出异能世界的存在”）。
5. 发展：列出关键发展事件（如主角寻找真相、拜师学艺、组建团队、遭遇挫折、与反派初步交锋等），说明事件之间的衔接逻辑。
6. 高潮：明确故事的核心高潮（如主角与反派的最终对决、真相揭晓的关键时刻、主角面临重大抉择等），要求高潮情节具备冲击力，解决核心矛盾。
7. 结局：确定结局类型（如圆满结局、悲剧结局、开放式结局、反转结局），说明结局的核心走向（如主角实现目标、人物获得成长、留下新的悬念等）。
8. **情节禁忌**：列出禁止出现的情节（如逻辑漏洞、人物人设崩塌、强行降智、俗套套路滥用、违背核心主题的情节等），例：“禁止出现主角明明有能力却故意放弃，导致剧情强行拖沓的情节”。
9. **细节要求**：明确是否需要加入特定细节（如环境描写、心理描写、动作描写、对话风格、时代/世界观特色细节等），例：“科幻小说需注重机甲、空间站的细节描写，体现硬科幻质感；历史小说需准确还原时代服饰、礼仪、语言习惯”。

## 五、风格与语言要求

1. **整体风格**：明确小说的整体基调（如严肃、幽默、伤感、热血、悬疑、温馨、暗黑等），例：“都市甜宠文需保持温馨甜蜜的风格，避免过多虐心情节；悬疑小说需营造紧张、压抑的氛围”。
2. **语言风格**：
        语言质感：确定语言的整体质感（如简洁凝练、细腻优美、通俗易懂、华丽辞藻、口语化、书面化等），例：“青春校园文可采用口语化语言，贴近青少年日常；历史小说需使用半书面化语言，体现时代厚重感”。
3. 对话风格：明确人物对话的特点（如符合人物身份、简洁有力、充满幽默感、暗藏玄机等），例：“反派的对话需充满城府，常使用暗示性语言；主角的对话需体现其性格，如乐观的主角对话轻松活泼”。
4. **描写要求**：
       描写侧重点：明确重点描写的维度（如心理描写侧重人物内心活动，环境描写侧重氛围营造，动作描写侧重画面感等）。
5. 描写禁忌：禁止出现的描写（如过于血腥暴力的细节、低俗色情的描写、冗长无意义的描写、与风格不符的描写等）。

## 六、其他补充要求

1. 是否需要加入特定元素（如彩蛋、伏笔、反转、致敬经典的情节、时代热点元素等）。
2. AI创作过程中的限制（如禁止擅自修改核心设定、禁止新增未提及的人物/情节、若出现逻辑问题需及时标注并提供修改方案等）。
3. 输出形式要求（如按章节输出、每章字数范围、是否需要附带情节梗概/人物小传等）。
4. 其他特殊需求（如避免使用特定词汇、适配特定平台的创作规范、符合相关审核要求等）。
